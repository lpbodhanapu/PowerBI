# Sales Performance Dashboard (Power BI)

## Overview
This project is a Power BI dashboard created using a real-world sales dataset.
The goal of this dashboard is to analyze sales performance, order trends, and
product distribution across time and countries.

The project covers the complete Power BI workflow:
- Data loading
- Data cleaning using Power Query
- Basic DAX measures
- Interactive dashboard creation

## Dashboard Highlights
- Total Sales, Total Orders, and Total Quantity KPIs
- Monthly order trend analysis
- Top selling products contribution
- Orders distribution across years
- Country-wise product sales distribution

## Tools Used
- Power BI Desktop
- Power Query
- DAX (basic measures)
- Microsoft Excel (CSV dataset)

## Dataset
- Name: Sample Sales Data
- Source: Kaggle
- Format: CSV
- Type: Retail sales transaction data
- Usage: Learning and portfolio project
